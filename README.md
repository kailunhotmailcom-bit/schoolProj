# My Blog - Personal Blog Platform

一个基于 Supabase 的个人博客系统，支持用户注册登录、发表文章和评论。

## 功能特性

- ✨ 用户注册和登录
- 📝 创建和浏览博客文章
- 💬 文章评论系统
- 🎨 现代化的响应式设计
- 🔒 基于行级安全策略（RLS）的权限控制

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **后端服务**: Supabase (PostgreSQL + Auth)
- **部署**: GitHub Pages 或其他静态托管平台
